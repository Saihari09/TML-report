How it works

  1. Pick a generator (Blood / HumanTrak / Wellbeing).
  2. Upload the appropriate file or fill the form manually. Use the "Load sample" button to try it without real data.
  3. Review the parsed values — you can edit anything before generating.
  4. Click Generate styled report. The report renders below the form.
  5. Click Print / Save as PDF to export. Browser print dialog → "Save as PDF".

Status colour key

GREEN — Normal
Within reference range. Good function, no immediate work limitation.
YELLOW — Mild Variance
Borderline value. Minor functional impact. Monitor and add targeted support.
ORANGE — Significant
Clear deviation from norm. Early functional limitation. Structured intervention recommended.
RED — Urgent
Marked deficit or asymmetry. High risk of progression or injury. Mandatory clinical action.
