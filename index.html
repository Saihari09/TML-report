Status colour key

GREEN — Normal
Within reference range. Good function, no immediate work limitation.
YELLOW — Mild Variance
Borderline value. Minor functional impact. Monitor and add targeted support.
ORANGE — Significant
Clear deviation from norm. Early functional limitation. Structured intervention recommended.
RED — Urgent
Marked deficit or asymmetry. High risk of progression or injury. Mandatory clinical action.

How it works

  1. Open the Unified Generator.
  2. Pick the patient's package (Blood & Movement Analysis / Comprehensive Movement Plan / Holistic Wellness).
  3. Drop the blood and HumanTrak PDFs — both auto-detected and parsed.
  4. Fill the Dynamo strength values + Nutri Meter / PSS-10 / PSQI manually (these are short).
  5. Click Generate. The report renders below; use Print → Save as PDF to export.
